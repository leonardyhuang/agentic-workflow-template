# Glossary

Centralized definitions for domain-specific terms, acronyms, and concepts. Update this when introducing new terminology.

## Format

```
## TERM
**Definition**: One-sentence definition.
**Where used**: Files, modules, or user flows where this term appears.
**Related**: Links to other glossary terms or docs.
```
