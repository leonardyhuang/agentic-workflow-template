# Technical Debt

Unfixed findings from code reviews, domain audits, or targeted sweeps go here before the session ends.
Verbal acknowledgement in conversation is not sufficient — write it down.

**Backlog** (features, improvements) → project management tool (Linear, GitHub Issues, etc.)
**Technical debt** (known code quality gaps, missing tests, process holes) → this file

## Format

```
## [HIGH|MEDIUM|LOW] Short title
- **File**: path/to/file.js:line
- **Finding**: what's wrong
- **Fix**: what needs to be done
- **Found**: YYYY-MM-DD
```

When items are fixed, remove them.
